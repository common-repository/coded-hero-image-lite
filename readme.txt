This plugin allows you to add a full width hero image onto your wordpress website and customize it to your liking.

== Description ==

Adding a hero image to your WordPress website can be a painful process especailly if you are not fluent in writing php and css.

This plugin was written with divi in mind and works perfectly for Divi theme users.

Detailed documenatation for this plugin can be found at https://coded.co.nz/documentation

== Installation ==

To install this plugin follow the steps below.


1. Upload the Coded Hero Image plugin folder (unziped) to the `/wp-content/plugins/` directory or upload the zipped version via Wordpress
2. Activate the plugin through the 'Plugins' menu in WordPress

== Screenshots == 

1.Coded Hero image lite allows you to quickly add a hero image to almost any WordPress theme (including Divi). Simply add an image, add an optional title and hit save.
